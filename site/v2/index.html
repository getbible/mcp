Required cache integrity

Store every cached scripture payload with the hash for its exact scope. Revalidate at least weekly. When a hash changes, invalidate that scope and every cached descendant, then atomically replace the stale content.

This synchronization cycle is a condition of the GetBible API usage agreement. Preserve the copyright metadata returned for each translation.
